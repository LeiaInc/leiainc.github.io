Use this method to perform all graphics rendering - interlacing, sharpening, and GUI display. This method is similar to DoPostProcess() but allows you to pass in the views to interlace and the destination render-target.

Parameters
[in]widthWidth of the render-target.
[in]heightHeight of the render-target.
[in]pictureResourceViews texture atlas.
[in]frameBufferDestination frame buffer for interlaced output.
[in]colorImageDestination frame buffer color image.
[in]depthImageDestination frame buffer depth image.
